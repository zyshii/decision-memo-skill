# Full Memo Template

Use this structure when producing a Full Mode decision memo. The template is rendered in Notion-flavored Markdown. Chinese is the primary language for Part A; Part B is bilingual.

---

## 元信息

- **决策 ID**：`[PREFIX]-NNN`
- **项目**：[Project name]
- **决策重要性**：🔴 High / 🟡 Medium / 🟢 Low
- **一句话概括**：[≤ 15 字]
- **我的角色**：[决策者 / 方案提议者 / 执行者 / 观察者]
- **状态**：[已完成 / 搭建中 / 进行中 / 暂停]

> ⚠️ **记录视角声明**（可选，仅当决策容易被误读为某个具体动作时）：说明这个 memo 的核心决策是什么，避免读者（包括未来的自己）误把"工具选择"当"策略判断"。

## 3. Stakeholder 动态（原始版）

- **支持方**：[谁、立场、动机]

- **中立方**：[若相关]
- **我的推进动作**：[如何说服、如何包装、如何避免正面冲突]
- **关键对话节点**：[具体会议/消息的回忆]
- **未解决的张力**：[哪些分歧没真正消化，只是搁置]
- **政治判断**：[做了哪些"非设计/非专业"的权衡，例如保留关系、节省资本、选择战场]

## 4. 决策过程（Process）

- **方法**：[数据驱动 / 用户研究 / 启发式 / 直觉 / 模式识别 / 混合]

- **拍板人**：[我 / PM / 老板 / 共识]
- **我的角色边界**：[我是决策者、提议者还是执行者]
- **如果重来一次我会怎么改方法**：[不是改决策，而是改达成决策的过程]
